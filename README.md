# BadAppleButInRoblox
Play bad apple in roblox studio!


How to set up:

1. Open script.lua and copy the contents.
2. Open a roblox studio project and create a script in workplace.
3. Paste the contents of script.lua into this script that we just made.
4. Open BadAppleData.lua and copy the contents.
5. Create a ModuleScript in roblox studio as a child of the script we made before.
6. empty the contents of the ModuleScript and rename it to "BadAppleData_32".
7. Now paste the contents of BadAPpleData.lua into BadAppleData_32
8. It is normal for studio to freeze at the pasting step as this file is over 6000 lines long.
9. Once unfrozen press play and touch the button with your character.

It should now play.
Enjoy!
